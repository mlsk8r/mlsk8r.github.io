Show and Hide Hidden Content with jQuery

a tutorial for beginners in jQuery


With some planning and a fairly simple setup, you can use jQuery to quickly add and remove CSS classes to hide and show content. View the demo below to see the final results of this turtorial.

VIEW DEMO

*This tutorial assumes the reader has a working knowledge of HTML and CSS

The HTML Setup

We first want to setup our base HTML file. You will have visible page content and hidden page content. You will need to create seperate classes for the div overlay that is hidden and for the "open" and "close" buttons.


        <!doctype html>
          <html lang="en">
          <head></head>
          <body>
          <!--create the div overlay box-->
          <div class="div-overlay">
            <!--Creates the div overlay content-->
            <div class="div-content-overlay">
              <h2>I'm a hidden div that is now visible</h2>
              <p> This is done using jQuery to add and remove css classes.</p>
              <!--the close button-->
              <button class="div-overlay-close">Close Me</button>    
            </div>
          </div>
          <!--page content when the div overlay is hidden-->
          <h2>Example of How jQuery Can Show & Hide Content</h2>
          <button class="div-overlay-open">More Information</button>
          </body>
        </html>

We will use the open and close buttons and their classes to trigger the hidden content to be visible with jQuery selectors.


The CSS Setup

You need to use CSS to hide the div-overlay and div-content-overlay elements. That way they are hidden by default on page load and later we can use jQuery to show the content by adding a new class. We hide the overlay content by changing the "visiblity" attribute to "hidden":


      .div-overlay {
        visibility: hidden;
      }

      .div-content-overlay {
        visibility: hidden;
      }

We don't want the hidden overlay content to be permanantly hidden. We want it to be visible when triggered by a button click.

To do this we use an ability of jQuery to add and remove a CSS class called "active" when the open and close buttons are clicked. We create this CSS class in our style sheet ahead of time so that it's ready to be used when the jQuery adds or removed it.

The example below shows our "div-overlay" and "div-content-overlay" classes with a new class of ".active" appended. We set the visiblity to "visible" now that we want them both to be active and seen.


      .div-overlay.active {
        visibility: visible;
      }

      .div-content-overlay.active {
        visibility: visible;
      }

The jQuery Setup

Adding the jQuery Library

First make sure you have the jQuery library linked to your HTML file.

You can do this similar to how you would link to a CSS file.

This first code snippet is a link to the jquery-3.3.1.min.js file tht you can download and store locally in your project folder.


      <script src="(filepath)/jquery-3.3.1.min.js" ></script>

This second code snippet is a way to link to the jquery file directly from the jquery code CDN.


      <script src="https://code.jquery.com/jquery-3.3.1.min.js" ></script>

It's up to you which way you want to access the jQuery library. Depending on your goals, either way works fine.

The jQuery code

We talked earlier about the ability jQuery has to add and remove a CSS class. We want to add and remove the "active" class we created earlier in the CSS setup when the open and close buttons are clicked.


      .div-overlay.active {
        visibility: visible;
      }

      .div-content-overlay.active {
        visibility: visible;
      }

Each of the statements below use the jQuery "on-click" method.

The first jQuery statement appends the "active" class to our hidden content when the "open" button is clicked. This will trigger our "active" css class and make the hidden content visible. The second jQuery statement will do the opposite of the first. It will remove the appended "active" class from the now visible content. This will make the content hidden again, or not visible.

Take a minute to look over the jQuery code and see if you can understand how it it works and what the code is doing.


      //appends an "active" class to .div-overlay and .div-content-overlay when the button is clicked
      $(".div-overlay-open").on("click", function(){
        $(".div-overlay, .div-content-overlay").addClass("active");
      });


      //removes the "active" class when the "Close Me" button is clicked 
      $(".div-overlay-close").on("click", function(){
        $(".div-overlay, .div-content-overlay").removeClass("active");
      });

Conclusion

The are many ways to use jQuery to hide and show content. This is just one example of how to do this by adding and removing CSS classes when the jQuery code is triggered.

I encourage you to explore other ways this type of effect can be achieved and learn more about what jQuery can do. Happy coding!
