Show and Hide Hidden Content with jQuery

a tutorial for beginners in jQuery


With some planning and a fairly simple setup, you can use jQuery to quickly add and remove CSS classes to hide and show content. View the demo by clicking the button below to see the final results of this turtorial.

VIEW DEMO

*This tutorials assumes that the reader has a working knowledge of HTML and CSS

The HTML Setup

The first thing we need to setup is our HTML document.

You will have visible page content and hidden page content. You will need to create seperate class for the div overlay that is hidden and for the open and close buttons.


        <!doctype html>
          <html lang="en">
          <head></head>
          <body>
          <!--create the div overlay box-->
          <div class="div-overlay">
            <!--Creates the div overlay content-->
            <div class="div-content-overlay">
              <h2>I'm a hidden div that is now visible</h2>
              <p> This is done using jQuery to add and remove css classes.</p>
              <!--the close button-->
              <button class="div-overlay-close">Close Me</button>    
            </div>
          </div>
          <!--page content when the div overlay is hidden-->
          <h2>Example of How jQuery Can Show & Hide Content</h2>
          <button class="div-overlay-open">More Information</button>
          </body>
        </html>

We will use the open and close buttons and their classes to trigger the hidden content to be visible with jQuery selectors.


The CSS Setup

You need to use CSS to hide the div-overlay and div-content-overlay elements. That way they are hidden by default on page load and later we can use jQuery to show the content by adding a new class. We hide the overlay content by changing the "visiblity" attribute to "hidden":


      .div-overlay {
        visibility: hidden;
      }

      .div-content-overlay {
        visibility: hidden;
      }

We don't want the hidden overlay content to be permanantly hidden. We want it to be visible when triggered by a button click.

To do this we use and ability of jQuery to add and remove a CSS class called "active" when the open and close buttons are clicked. We create this CSS class in our style sheet ahead of time so that it's ready to be used when the jQuery adds or removed it.

The example below shows our "div-overlay" and "div-content-overlay" classes with a new class of ".active" appended. We set the visiblity to "visible" now that we want them both to be active and seen.


      .div-overlay.active {
        visibility: visible;
      }

      .div-content-overlay.active {
        visibility: visible;
      }

The jQuery Setup


      //appends an "active" class to .div-overlay and .div-content-overlay when the button is clicked
      $(".div-overlay-open").on("click", function(){
        $(".div-overlay, .div-content-overlay").addClass("active");
      });

      //removes the "active" class when the "Close Me" button is clicked 
      $(".div-overlay-close").on("click", function(){
        $(".div-overlay, .div-content-overlay").removeClass("active");
      });
