Program Finder UX Research and Development

Southern Utah University

Project Overview

Desktop View

TLDR

Research the user experience of the SUU program finder webpage

Plan and conduct user testing with a mixed-methods approach to acquire both qualitative and quantitative data

Develop conclusions and recommendations based on user testing results and data

Present reports based on the final results of user testing and data analysis

Implement recommendations into development and production

The Project

The SUU program finder tool and webpage were developed and designed without any real-world user testing before launch. In order to improve the user experience and overall function of the program finder webpage, user testing was initiated with the following goals.

  • Understand what works and what does not work on the Program Finder web page by conducting user tests
  • Have the participants attempt real-world activities and complete realistic goals in order to gain qualitative and quantitative insights into how the web page is being used
  • Use these insights and data to help determine how to improve the user experience
  • Present conclusions and recommendations to those who can start implementing improvements into production

After initial planning and review of related literature, the following research objectives were developed and refined for the user tests:

  • Discover if the Program Finder is actually helping current and prospective students learn about programs and degrees of interest
  • Explore new options for gathering and presenting relevant program information based on the interests of the user
  • Understand how the average user feels when they are exploring and using the Program Finder
  • Identify different user approaches to a given task from start to finish by observation and use of the Program Finder

To write effective tasks scenarios for user testing, a list of general user goals that visitors to the site want to accomplish was developed. In order to create those user goals, the following question was asked, “What are the most important things that every user must be able to accomplish when using the Program Finder?”

The following goals were developed based on that question:

  • Find specific programs and degrees of interest
  • Narrow down options based on interest and degree type by using the filter tools
  • Find relevant programs based on initial searching and filtering
  • Read useful information about a program of interest found without leaving the program finder web page
  • Find out more information beyond the program finder web page about a degree or program of interest by using external links
  • Search degrees effectively by category filtering

After research, planning, and facilitating the user tests, valid and valuable conclusions and recommendations were developed based on gathered user data and feedback both qualitative and quantitative in nature (mixed-methods data).

A report and presentation was produced and presented to stakeholders and departments that had a vested interest in the improvement of the Program Finder webpage. The report and recommendations were received with enthusiasm and optimism for improving the user experience. Approval was given to move forward with implementing many of the recommendations into production to improve the user experience of the application.

Improvements to the Program Finder webpage were made both visually and functionally based on the user testing data and report. Working in collaboration with developers, the Program Finder website was approved for a Phase II launch with the new improvements based on the outcomes of the project.
