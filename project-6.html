TLDR

Collaborate with graphic designers about ideas and concepts for better user engagement and experience for calls to action

Develop prototypes in Adobe XD for better pre-visualization

Develop rapid web prototypes to showcase quick ideas and animations

Present prototypes and designs to the creative group and receive constructive feedback and input on how to improve new features and designs

The Project

As part of the overall redesign and rebranding of the SUU website, tasks were given to design, prototype, and develop interactive user features. These new interactive features would promote better user engagement and facilitate user action.

Designs were created and presented that would create web elements with a feeling of depth, dimension, and motion that still images couldnâ€™t produce on their own.

The overall goal of these engaging elements is to draw user attention to calls to action and promote user action.

Designs were presented as mockups and then as prototypes in using Adobe Premiere Pro to show animation. Later designs were translated and developed into rapid web prototypes. These prototypes were shared with the creative team and presented as options for new elements to incorporate into the overall new design of the SUU website. The prototypes and concepts were received with enthusiasm and will be used in the new redesigns of the SUU website in the near future.
